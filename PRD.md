# PRD: CX 회의록 & 보고서 자동화 도구

## 1. 프로젝트 개요

CX팀의 회의 내용을 AI로 자동 정리하고, 축적된 데이터를 기반으로 주간/월간 보고서를 자동 생성하는 웹 앱.

## 2. 배경 및 문제 정의

CX팀 팀장은 매일 반복되는 두 가지 업무에 가장 많은 시간을 소비하고 있다:

- **회의록 정리**: 회의를 음성 녹음한 뒤, 녹음을 다시 듣고 텍스트로 정리하는 과정이 회의 시간만큼 또는 그 이상 소요된다.
- **주간/월간 보고서 작성**: CX 지표(문의 건수, 처리 현황 등)를 수기로 집계하고, 이를 보고서 양식에 맞춰 작성하는 작업이 매주/매월 반복된다.

이 두 업무는 구조화된 패턴이 있어 AI 자동화에 적합하며, 자동화 시 업무 시간을 절반 이하로 줄일 수 있다.

## 3. 목표 사용자

| 사용자 | 역할 | 주요 사용 기능 |
|--------|------|---------------|
| CX팀 팀장 (Primary) | 회의록 정리, 보고서 작성 및 관리 | 회의록 자동 정리, 보고서 생성, 데이터 입력 |
| CX팀 팀원 | 회의록 확인, 액션 아이템 관리 | 회의록 조회, 자신의 액션 아이템 확인 |
| 상급 관리자 | 보고서 수신 및 현황 파악 | 보고서 조회, CX 현황 대시보드 확인 |

## 4. 핵심 기능 (우선순위별)

### P0 — MVP 필수

**4.1 음성 녹음 및 텍스트 변환**
- 브라우저에서 직접 회의 음성 녹음 (녹음 시작/중지 버튼)
- Web Speech API를 활용한 실시간 음성→텍스트 변환 (STT)
- 변환된 텍스트 확인 및 수동 수정 가능
- 외부에서 변환된 텍스트를 직접 붙여넣기도 지원

**4.2 회의록 자동 정리**
- 변환/입력된 텍스트를 기반으로 AI가 자동 구조화:
  - 회의 기본 정보 (일시, 참석자, 주제)
  - 안건별 논의 내용 요약
  - 결정사항 목록
  - 액션 아이템 (담당자, 기한, 내용)
- 결과 미리보기 및 수동 수정 가능
- 완성된 회의록 저장

**4.3 주간 보고서 자동 생성**
- 기간 선택 (이번 주, 지난 주, 커스텀)
- 해당 기간 회의록 + CX 지표 데이터 자동 취합
- 보고서 템플릿 기반 초안 자동 생성
- 보고서 내용:
  - 기간 내 주요 활동 요약
  - CX 지표 현황 (문의 건수, 유형별, 처리율)
  - 주요 이슈 및 특이사항
  - 다음 주 계획
- 미리보기, 수정, 다운로드(마크다운/복사)

### P1 — 우선 확장

**4.4 CX 지표 입력/관리**
- 문의 건수 입력 (유형별: 전화, 이메일, 채팅 등)
- 처리 현황 입력 (해결, 미해결, 진행 중)
- 주요 이슈 및 특이사항 텍스트 입력
- 입력 데이터 저장 및 이력 조회

**4.5 회의록 이력 관리**
- 저장된 회의록 목록 조회
- 회의록 검색 (날짜, 키워드)
- 회의록 상세 보기

### P2 — 향후 확장

**4.6 월간 보고서 자동 생성**
- 월간 단위 데이터 취합 및 보고서 생성
- 월간 트렌드 분석 포함

**4.7 데이터 시각화**
- CX 지표 추이 차트
- 문의 유형별 분포 그래프

## 5. 비기능 요구사항

- **접근성**: 웹 브라우저에서 바로 사용 가능 (별도 설치 불필요)
- **반응형**: 데스크톱과 모바일 모두 사용 가능
- **사용성**: 비개발자도 즉시 사용 가능한 직관적 UI
- **성능**: 회의록 정리 결과가 10초 이내에 생성
- **데이터 저장**: 브라우저 로컬 또는 간단한 파일 기반 저장

## 6. 성공 지표

| 지표 | 현재 | 목표 |
|------|------|------|
| 회의록 정리 소요 시간 | 30~60분/건 | 5~10분/건 |
| 주간 보고서 작성 시간 | 1~2시간 | 15~30분 |
| 보고서 양식 일관성 | 작성자마다 다름 | 100% 통일 |
| 회의 액션 아이템 누락률 | 수기 정리로 누락 발생 | AI 추출로 누락 최소화 |

## 7. 제약 사항 및 가정

### 제약 사항
- 워크숍 실습 프로젝트로, 실제 프로덕션 수준의 보안/인프라는 고려하지 않음
- 음성→텍스트 변환은 브라우저 Web Speech API를 활용하며, 한국어 인식 정확도는 브라우저/환경에 따라 다를 수 있음
- AI API 연동 시 API 키 관리 필요

### 가정
- 사용자는 웹 브라우저를 사용할 수 있음
- 회의 녹음의 음성→텍스트 변환은 이미 완료된 상태로 텍스트를 입력
- CX 지표 데이터는 사용자가 직접 입력

## 8. MVP 범위

**MVP에 포함:**
- 브라우저에서 직접 음성 녹음 및 실시간 텍스트 변환 (Web Speech API)
- 외부 텍스트 붙여넣기도 지원
- 회의 텍스트 → AI 자동 정리 (안건, 결정사항, 액션아이템)
- 정리된 회의록 미리보기 및 수정
- CX 지표 간단 입력 폼
- 주간 보고서 자동 생성 (회의록 + CX 지표 기반)
- 보고서 미리보기 및 복사/다운로드

**MVP에서 제외 (향후 확장):**
- 회의록 이력 관리 및 검색
- 월간 보고서
- 데이터 시각화/차트
- 사용자 인증/권한 관리
- 외부 시스템 연동 (캘린더, CRM 등)
