# CX팀 회의록 + 보고서 자동화 도구 — 아이디어 문서

## 해결하려는 문제

CX팀 팀장으로서 매일 반복하는 업무 중 가장 시간이 많이 걸리는 것:

1. **회의 내용 정리**: 음성 녹음을 듣고 회의록으로 정리하는 작업
2. **주간/월간 보고서 작성**: CX 데이터를 수기로 집계하고 보고서를 작성하는 작업

이 두 가지 업무는 반복적이면서 시간 소모가 크고, 구조화된 패턴이 있어 AI 자동화에 적합하다.

## 선택한 해결 방안

**방안: AI 회의록 자동 정리 + 보고서 생성 통합 도구**

회의 텍스트(음성→텍스트 변환 결과)를 입력하면 자동으로 구조화된 회의록을 만들고, CX 지표 데이터를 축적하여 주간/월간 보고서까지 자동 생성하는 웹 앱.

### 장점
- 회의록 작성과 보고서 생성을 하나의 워크플로우로 통합
- CX 지표 자동 추적 가능
- 비개발자도 바이브 코딩으로 구현 가능

### 단점
- 초기에 보고서 양식과 CX 지표 정의 필요
- 음성→텍스트 변환은 외부 서비스(클로바노트 등) 활용 필요

## MVP 범위

### Phase 1 (MVP)
- 회의 텍스트를 붙여넣으면 → 구조화된 회의록 자동 생성
  - 참석자
  - 안건별 논의 내용
  - 결정사항
  - 액션 아이템 (담당자, 기한)
- CX 수기 지표 입력 폼
- 주간 보고서 초안 자동 생성

### Phase 2 (확장)
- 월간 보고서 자동 생성
- 데이터 누적 및 트렌드 분석
- 이전 회의 검색 기능

## 필요한 기능 목록

1. **회의록 자동 정리 기능**
   - 텍스트 입력 영역
   - AI 기반 구조화 (안건, 결정사항, 액션아이템 분류)
   - 회의록 미리보기 및 수정
   - 회의록 저장 및 이력 관리

2. **CX 지표 입력 기능**
   - 문의 건수 (유형별: 전화, 이메일, 채팅 등)
   - 처리 현황 (해결, 미해결, 진행 중)
   - 주요 이슈 및 특이사항 입력

3. **보고서 자동 생성 기능**
   - 주간 보고서 템플릿 기반 자동 생성
   - 기간 선택 (이번 주, 지난 주 등)
   - 보고서 미리보기 및 수정
   - 다운로드 (마크다운, 복사)

4. **데이터 관리**
   - 회의록 이력 목록
   - CX 지표 데이터 저장
   - 간단한 통계/차트

## 기술적 실현 가능성

- **구현 방식**: 웹 앱 (HTML + JavaScript + AI API)
- **바이브 코딩 적합도**: 높음 — 자연어로 지시하여 구현 가능
- **추천 기술 스택**: React 또는 단일 HTML, Tailwind CSS, AI API 연동
- **난이도**: 중간
- **예상 구현 시간**: 바이브 코딩으로 MVP 1~2일

## 현재 데이터 환경

- **회의 기록**: 음성 녹음 → 텍스트 변환 후 정리
- **CX 지표**: 수기 집계
- **보고서 양식**: 기존 주간/월간 보고서 포맷 존재
