# 개발 계획: CX 회의록 & 보고서 자동화 도구

## 기술 스택

| 영역 | 기술 | 선택 이유 |
|------|------|-----------|
| 프론트엔드 | HTML + Tailwind CSS + Vanilla JS | 빌드 도구 없이 바로 실행 가능, 비개발자 유지보수 용이 |
| AI 연동 | OpenAI API (또는 Claude API) | 회의록 구조화 및 보고서 생성용 |
| 음성→텍스트 | Web Speech API | 브라우저 내장, 무료, 별도 서버 불필요 |
| 데이터 저장 | IndexedDB (브라우저 내장) | 서버 없이 로컬 저장, 데이터 영속성 보장 |
| 배포 | GitHub Pages 또는 정적 호스팅 | 무료, 간편한 배포 |

## MVP vs 향후 확장

### MVP (Phase 1)
- 프로젝트 기본 구조
- 음성 녹음 + 텍스트 변환
- AI 회의록 자동 정리
- CX 지표 입력
- 주간 보고서 자동 생성
- 보고서 복사/다운로드

### 향후 확장 (Phase 2~3)
- 데이터 로컬 저장 (IndexedDB)
- 회의록 이력 관리 및 검색
- 월간 보고서
- 데이터 시각화 차트
- 모바일 최적화
- 배포
